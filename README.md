## 概要

Django REST Frameworkを使用したWebAPIサンプルプロジェクトです。

## 機能

- Django REST Framework によるAPI構築
- PostgreSQL / SQLite データベース対応
- Redis キャッシュシステム
- Docker コンテナ化対応
- CORS設定済み
- Whitenoise による静的ファイル配信
